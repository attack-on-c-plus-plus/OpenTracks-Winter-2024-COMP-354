v4.9.0: OpenTracks

ВНИМАНИЕ: СУЩЕСТВЕННОЕ ИЗМЕНЕНИЕ

В этой версии меняется порядок определения OpenTracks состояния движения.
До настоящего времени скорость (от GPS или датчика) использовалась с применением порогового значения (например, менее 0,3 м/с).
Однако, это может приводить к получению очень "шумных" ("грязных") данных.
Теперь же используется длительность таймаута (т.е. период времени без существенных изменений местоположения длительностью X секунд).
Эта длительность задается пользователем.
События перехода в режим без движения сохраняются в базе данных, включая метки времени, поэтому статистика может быть пересчитана.

Возврат к предыдущей версии от этой не предусмотрен.
Пожалуйста, выполните резервную копию данных перед обновлением.

Изменения:
- определение того, что пользователь не двигается, на основе периода "без движения" вместо порога минимальной скорости
- добавлен вид активности "самокат"

Исправление ошибок:
- во время записи в списке треков индикация записи отображалась на всех треках

Для разработчиков:
- снова включены тесты выполнения действий Github
